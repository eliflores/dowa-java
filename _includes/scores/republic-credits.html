Credits were a standardized form of currency in use throughout the Galactic Republic and later the Galactic Empire.

From time to time we will hava little contests / challenges in groups of 1, 2 or 3 students. Each team will earn Republic Credits as follows:

  • 2 credits: if team answers / finishes an exercise first.
  • 1 credit: if team answers / finishes an exercise.
  • Java Jeopardy credits depend on the answered question.
This means that the team that does not answer / finish an exercise will not earn any credits.

Creds can be exchanged by different items only available in the DoWA Java Territories of the Galactic Republic :-)
