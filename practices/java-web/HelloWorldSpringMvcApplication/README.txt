how to run it:

mvn clean install
mvn tomcat7:run

webapp should be up and running on port 9090

go to:

localhost:9090/hello-spring-mvc/

Your task is:
1. Create a new handler method that will handle the post request in the 'who-are-you' form that was added in index.jsp
2. Display in the view (it could be a new view) the 'name' value that was sent in the post request
3. Create a MR with your code
