Overview

Este curso les compete a alumnos que se encuentran en la etapa disciplinaria de la Licenciatura en Ciencias Computacionales y que desean conocer y aprender a utilizar tecnologías que están ampliamente difundidas en el mercado actual de los sistemas de nformación. Al concluir este curso, los alumnos deben de estar capacitados para llevar a cabo las siguientes tareas:
  • Explicar la base teórica que clarifique los conceptos abarcados por la plataforma Java.
  • Aplicar la programación orientada a objetos usando el lenguaje de programación Java.
  • Decidir qué elementos de la plataforma Java son los más indicados para llevar a cabo un desarrollo de una aplicación Web de moderada complejidad.
  • Depurar aplicaciones Web en Java.
  • Implementar una aplicación Web desarrollada en Java dentro de un servidor de aplicaciones.
  • Identificar los principios de diseño más usados para el desarrollo de una aplicación Web.

Competencias del Curso

Desarrollar aplicaciones Web con el uso de tecnologías y lenguaje de programación Java aprendiendo a identificar, manipular y aplicar las herramientas, guías y componentes más utilizados para su diseño e implementación en un servidor de aplicaciones.

Criterios de evaluación

  • Primer examen parcial: 20%
  • Segundo examen parcial: 20%
  • Prácticas y tareas: 30%
  • Proyecto final y exposición: 30%
